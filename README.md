## 🏗️ Proyectos en Producción

### 🏥 Sistema de Gestión de Informes Médicos
> Plataforma fullstack para profesionales de la salud con panel médico y portal de pacientes 24/7

- **-80% en tiempo de creación de informes** → de 15 min a 3 min por informe
- Plantillas reutilizables por tipo de estudio (Ecografía, TAC, Doppler)
- Generación automática de PDFs con firma digital · Almacenamiento en AWS S3
- Portal del paciente: acceso por DNI, descarga de estudios sin ir al consultorio
- **Stack:** Next.js 16 · React 19 · TypeScript 5 · Tailwind CSS 4 · Zustand · pdfmake · Prisma · PostgreSQL · AWS S3

---

### 🛒 TuTiendaWeb — SaaS para Gestión de Comercios
> Plataforma multi-tenant para que comercios gestionen productos, ventas y clientes

[![Live](https://img.shields.io/badge/🌐_Ver_en_vivo-tutiendaweb.com.ar-E8550A?style=flat-square)](https://tutiendaweb.com.ar)

- +3 comercios activos en producción con ventas reales
- Catálogo online · Menú QR · Sistema de pedidos integrado con WhatsApp
- Integración MercadoPago para suscripciones recurrentes
- Reportes financieros automatizados en PDF y Excel
- **Stack:** Next.js · TypeScript · Firebase/Firestore · Tailwind CSS · Vercel

- **Stack:** n8n · WhatsApp Business API · Google Sheets API
